# AI Helpdesk Automation Assistant

A beginner-friendly full-stack project built with React and Node.js.

## Tech Stack

- React
- Vite
- Node.js
- Express

## Project Structure

- `client/` - React frontend
- `server/` - Node.js backend

## Milestone 1

- Set up GitHub repository
- Created React frontend
- Created Node.js backend
- Confirmed both run locally

## How to Run

### Backend

```bash
cd server
npm install
npm start
```
